Create Project



A SAVR project is a mobile application and can contain an indefinite number of builds.



Your project will have an application ID and package ID. These will allow all builds of your project be accepted by app stores for a specific application.



Your mobile applications will be built in release mode and as such, will be signed. You will be emailed the signing keys upon successful build completion.


To create a project, follow these easy steps:



1 - Login to your SAVR dashboard and click on the Add project button.





2 - You will be asked to provide your application name (This name will appear to users who download your app and cannot be changed), developer name (This will be used when signing), a brief description, and an app icon (You may change this when creating a build).





3 - After providing these details, click on proceed and you're all set.
